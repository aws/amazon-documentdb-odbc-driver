All files are available in '/Library/ODBC/database-odbc' after installation.

To setup a connection, you can use DSN to store your data source connection information,
1. Open 'iODBC Data Source Administrator'.
2. Go to 'User DSN' and click 'Add'.
3. Select 'Database ODBC Driver' and click on 'Finish'.
4. Update the connection string values. For the list of all supported options, check '/Library/ODBC/database-odbc/doc/configuration_options.md' and '/Library/ODBC/database-odbc/doc/mac_configure_dsn.md'.
5. Click 'Ok' to save changes.

For information on using the Database ODBC Driver with Excel, see '/Library/ODBC/database-odbc/doc/microsoft_excel_support_mac.md'
